UX Portfolio

Mobile Apps for Android

Summer 2020

These projects were completed through the CodePath Android bootcamp for Facebook University software engineering interns. Click on each gif to view the project on Github.

Twitter App Mockup

Simple Tweet is an android app that allows a user to view their Twitter timeline and post new tweets directly to the actual Twitter app. The app utilizes Twitter REST API.

Instagram App Mockup

This Instagram mockup is a photo sharing app using Parse as its backend. It includes features such as endless scrolling, swiping to refresh the page, and usage of the phone's camera.

Flixster Movie App

Flixster shows the latest movies currently playing in theaters. The app utilizes the Movie Database API to display images and basic information about these movies to the user.

SageDorms

Spring 2020

Team: Gabriel Alzate, Yurie Muramatsu, Helen Paulini, Ilana Shapiro, Alan Zhou. Sagedorms is a virtual roomdraw system for Pomona College. This app was created and built as a semester long team project in CS 133, a Databases course.
